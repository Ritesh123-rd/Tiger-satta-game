# Assets Information

This folder should contain the following files for a complete Expo app:

1. icon.png (1024x1024 px) - App icon
2. splash.png (1242x2436 px) - Splash screen
3. adaptive-icon.png (1024x1024 px) - Android adaptive icon
4. favicon.png (48x48 px) - Web favicon

Since we cannot generate images, please add your own images with these names and dimensions.

You can use any image editor or online tools to create these assets.

For now, the app will work with default Expo placeholders during development.
